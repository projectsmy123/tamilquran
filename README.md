# Tamil Qur'an Recitation Aid — Surah Al-Fatihah v7

Static GitHub Pages prototype for a mobile-first Tamil Qur'an recitation aid.

## v7 changes

- Beige background is now the default site theme.
- The full Arabic ayah line above the boxes has been removed.
- One ayah is shown per swipe page.
- Users move to the next ayah by swiping horizontally.
- Continuous Tamil transliteration wraps into multiple lines instead of being clipped.
- Arabic and Tamil sound units inside word boxes are rendered right-to-left.

## Existing features

- One box per Arabic word.
- Unique word-border color per word in each ayah.
- Arabic units and corresponding Tamil units share the same letter color.
- Tamil direct transliteration appears inside each word box.
- Tap a word box to play the word audio.
- Ayah play/stop buttons use Sheikh Mahmoud Khalil Al-Husary audio.
- Abdul Hameed Baqavi Tamil translation appears after continuous Tamil transliteration.

## GitHub Pages upload

Upload these four files directly into the repository root:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
