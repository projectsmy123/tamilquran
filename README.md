# Tamil Qur’an Reciter — Full Qur’an Engine Prototype v9

This is a GitHub Pages static web app.

## What this version does

- Supports all 114 surahs.
- Loads Arabic ayah and word data from the Quran Foundation API.
- Loads Abdul Hameed Baqavi Tamil translation from QuranEnc using the `tamil_baqavi` translation key.
- Generates Tamil direct transliteration with the project’s own rule-based Arabic → Tamil engine.
- Keeps one ayah per mobile page.
- Supports swipe left/right for previous/next ayah.
- Keeps Arabic words inside individual color-outlined boxes.
- Keeps Arabic and Tamil sound units right-to-left inside each box.
- Tap a word box to play that word audio.
- Play/Stop ayah recitation using Sheikh Mahmoud Khalil Al-Husary audio.

## Important

The Tamil transliteration is generated by our own first-pass rule engine. It is not copied from TamililQuran. It must be reviewed and improved surah by surah.
